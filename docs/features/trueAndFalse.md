# 真・偽
真・偽自体と、真か偽の判断

判断で、真の場合は`true`、偽の場合は`false`に置き換えます。
## 使い方
- 真
```
true
```
- 偽
```
false
```
一個目と二個目の値を、xとyに抽象化したり、意味のない変数をaとして、説明書きます。
- 等しいか
```
<x=y>
```
- xよりyが...
```
<x<y>
<x>y>
```
- 変数が存在するか
```
<a?>
```
## 使用例
```
# 5と5は同じか
<5=5>
# 1は2より小さいか
<1<2>
# 1は2より大きいか
<1>2>
# 変数hogeが存在するか
<hoge?>
```
## 拡張
```
# AND(論理積)
<<hoge=2><fugaga=2>=truetrue>

# OR(論理和)
<<<hoge=2><fugaga=2>=falsefalse>=false>
```
